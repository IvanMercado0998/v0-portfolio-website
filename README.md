## How It Works

1. Create or modify your project using **NAVICOM CloudSystems**.  
2. Deploy your updates directly from the NAVICOM platform.  
3. Changes are automatically pushed to this Git repository.  
4. Vercel automatically deploys the latest version from this repository, keeping your portfolio live and updated.
